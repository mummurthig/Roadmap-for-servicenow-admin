Full Syllabus

Module 01 · Foundation
Platform Fundamentals
  • ServiceNow architecture — instances, nodes, MID server
  • UI & navigation — Application Navigator, Service Portal
  • Tables, records, fields & data types
  • Forms, lists, views & related lists
  • Users, Groups, Roles & ACLs (Access Control)
  • Import Sets & Transform Maps
Module 02 · ITSM Core
IT Service Management
  • Incident Management — lifecycle, SLAs, escalation rules
  • Problem Management — root cause, workarounds
  • Change Management — CAB, types, risk assessment
  • Request Management — catalog items, workflows
  • CMDB — configuration items, relationships, discovery
  • Knowledge Management — articles, feedback loops
Module 03 · Automation
Workflow & Flow Designer
  • Legacy Workflow editor — activities, conditions
  • Flow Designer — triggers, actions, subflows
  • Business Rules — before/after/async, script basics
  • Script Includes — reusable server-side logic
  • Client Scripts & UI Policies — dynamic form behavior
  • SLA Definitions & Breach notifications
Module 04 · Development
Scripting & GlideScript
  • GlideRecord — CRUD operations in JavaScript
  • GlideSystem — gs.log, gs.getUser, utility methods
  • GlideDate, GlideDateTime — date manipulation
  • Scoped vs Global applications
  • REST API Integration — Inbound/Outbound
  • Web Services — SOAP, REST, JSON payloads
Module 05 · Service Catalog
Catalog & Portal Design
  • Catalog items, record producers, order guides
  • Variable types — reference, list, multi-choice
  • Catalog client scripts & variable sets
  • Service Portal — widgets, pages, themes (AngularJS)
  • Employee Center (Next Experience / Polaris)
  • Approval policies & delegation
Module 06 · Reporting
Analytics & Dashboards
  • Reports — list, bar, pie, heat map, trend
  • Dashboards — role-based, homepage vs performance
  • Performance Analytics — scores, indicators, breakdowns
  • Scheduled reports & email distribution
  • Filters & conditions — encoded query strings
  • Now Intelligence & AI-assisted insights
Module 07 · Advanced Admin
Platform Configuration
  • Update Sets — capturing & migrating customizations
  • Instance cloning strategy (Prod → Dev)
  • Notifications — email, SMS, push templating
  • Domain Separation — multi-tenant setups
  • Scheduled Jobs & Events registry
  • System Diagnostics & Instance health checks
Module 08 · SecOps & HRSD
Expanded Modules
  • HR Service Delivery — cases, COEs, journeys
  • SecOps basics — Vulnerability Response, SIEM
  • Customer Service Management (CSM) overview
  • GRC — risk, compliance, audit basics
  • ITOM — Discovery, Event Management overview
  • Now Platform roadmap — Vancouver, Washington releases
Module 09 · Interview & Job Prep
Placement Readiness
  • Top 50 ServiceNow Admin interview Q&A
  • Scenario-based questions — ITSM, automation, ACLs
  • Live instance practice — personal developer instance
  • Resume keywords — ATS optimization for SN roles
  • Mock admin tasks — timed exercises
  • Salary negotiation script for ₹15L target
